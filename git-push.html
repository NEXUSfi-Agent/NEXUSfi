Git Operations Guide

Since browsers cannot execute Git commands directly, please execute the following commands in your terminal or command prompt:

  1. Open a command prompt or terminal window
  2. Navigate to the project directory: cd /path/to/NEXUSfi
  3. Check Git status: git status
  4. Add all changes: git add .
  5. Commit changes: git commit -m "NEXUSfi V1.02"
  6. Push to GitHub: git push origin master

SSH Connection Configuration

If you need to configure SSH connection to GitHub, follow these steps:

  1. Generate SSH key: ssh-keygen -t ed25519 -f ~/.ssh/id_github_nexusfi -C "NEXUSfiNEXUSfi@163.com"
  2. Add public key to your GitHub account: Open ~/.ssh/id_github_nexusfi.pub file and copy its contents
  3. Visit GitHub SSH Key Settings and paste the public key
  4. Test connection: ssh -T git@github.com
